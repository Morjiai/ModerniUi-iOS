ModerniUi-iOS

ModerniUI-iOS is a updated version of iUI, the update was created to make creating web apps easier. iUI is a great framework but sadly, no longer updated. This was created for people who enjoy the framework, and want to see it come back. It currently isn't a fork of iUI so the user will have to modify a version of iUI 0.4.1 themselves, you can also use the installer if you are on Linux or Mac!
